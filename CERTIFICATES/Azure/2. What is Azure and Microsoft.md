# What is Microsoft?
- multinational computer technology corporation
	- makes software, phones, tablets, game consoles, cloud services, a search engine and more!
	- best known for **Windows OS**
- headquartered in Redmond, Washington

# What is Azure?
- Microsoft's Cloud Service Provider (CSP)

# Azure's Deployment Models
<hr>
## Public Cloud
- Everything built on Cloud Provider
- Everything running on "Azure"
- aka Cloud-Native
![[Azure Network Arch Diagram.png]]
## Private Cloud
- Everything built on company's datacenters
- aka On-Premise
- The cloud could be some opensource mimic, like OpenStack
![[Private Cloud Arch Diagram.png]]
## Hybrid
- Using both On-Premise and a Cloud Service Provider (connected together)
- Networking Services to facilitate connection b/w the 2
	- e.g. ExpressRoute (dedicated connection, like fibre-optic line running from on-prem datacenter to azure network)
