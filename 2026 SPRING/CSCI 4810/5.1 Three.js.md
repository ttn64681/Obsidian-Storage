##  The Big Three: Scene, Camera, Renderer
- **Scene**: Container that holds all your 3D objects, lights, etc.
- **Camera**: Defines the viewpoint - what you see and from where
- **Renderer**: Draws the scene to a \<canvas> element using WebGL

- orbit contorls



## Non-Module Approach (Script Tag)
- simplest way to get started: Include Three.ks w/ a regular script tag
1. download three.js (or three.mjs)
2. ```
   <script src="js/three.js"></script>
   <script>
   	// THREE is now a global var
   	const scene = new THREE.Scene();
   	// ...
   </script>
   ```
- Pros: simple, works from file system, works offline
- Cons: loads entire library even if you only need few features
	- Adds extras like (OrbitConsole)
