## Animation Loop:
- Creating an Animation Loop:
- 1. Clear the canvas or the area you want to update.
- 2. Draw the objects on the canvas.
- 3. Repeat these steps using a `draw()` function in a loop.
- Basic Method:
	- Use `setInterval(function, timeout)` to create a simple loop.
	- Note: `setInterval()` may not work consistently across all browsers.
- Modern Method:
	- Use `requestAnimationFrame()` for smoother, more reliable animations.
	- Advantages:
		- Better performance and smoother visuals.
		- Works well across different browsers.
		- Syncs with the screen’s refresh rate.
