# Handling Missing Data:
- Pandas uses "**sentinels**" to mark missing values/data in a dataset
- Mainly 2 Sentinels:
	- None:
		- Python object, used in object arrays
	- NaN

## None
- NoneType Object
- Used in Object/String data
- Raises TypeError
- None == None is True

## NaN
- Float type object
- Used in Numerical data
- Results in NaN (propagates)
- np.nan == np.nan is False
	- counts as different objects

- Experimental singleton missing value indicator: pd.NA
- >>> pd.NA
- \<NA>
- >>> True | pd.NA
- True
- True & pd.NA
- \<NA>
- >>> pd.NA != pd.NA
- \<NA>
- >>> pd.NA == pd.NA
- \<NA

## Operating on Null Values
### Detection and Removal
- isnull(), isna()
- notnull()	
- dropna(): Return filtered version of data
	- removes records/columns

- if you don't want to drop the values, you can fill them instead/replace
### Filling Missing Values
- fillna(): Return copy of data with missing values replaced
- Can fill with constant, forward-fill (ffil) or backward-fill (bfill)
	- ffil - copy last seen value to missing value position
	- bfill - copy next seen value to missing value position

# Combining Objects
- concat():
	- Merge mult Series or DataFrame obj along a shared index or col
	- `result = pd.concat([df1, df4], axis=1)`
		- missing vals filled w/ NaN
	- concat(join):
		- `result = pd.concat([df1,df4] axis=1, join='inner')
		- returns the overlapping axes/rows
- DataFrame.join():
	- Merge multiple DataFrame objects along the columns
- merge():
	- Combine two Series or Dataframe objects
