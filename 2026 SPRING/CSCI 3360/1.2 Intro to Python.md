## Operators

**Arithmetic**
- 5//3 -> *floor div* -> 1
- -5//3 -> -2
- **result of division is always a float**
- ** - exponent

**Boolean**
- True + Ture = 2
- False - 5 = -5

- 0 == False -> True
- 2 > True -> True

**Relational**
- 1 == 1 -> True
- 2<3 and 3<2 -> False

**Identity**
- is vs ==
	- is: checks **if 2 vars refer to same obj**
	- \==: checks if objects pointed to have same value
```
a = [1]
b = a
b is a # True
b == a # False

b = [1]
b is a # False
b == a # True
```

**Membership**
- a = [1,2,3,4]
- 3 in a -> O(n)

- b = {1,2,3.4}
- 3 in a -> O(1)

## List
- li = [1,2,4,3]
- li.append(1) # appends to end, O(1)
- li.pop() # returns last val, O(1)

- li[0]
- li[-1] # reverse order
- li[4] # if out of bounds, raises IndexError

**Slicing**
- closed/open range
	- li[1:3]
	- li[2:]
	- li[:3] # from beg until ind 3
	- li[::2] # return list selecting elements w/ step size 2 => [1,4]
		- every other element starting w/ first
	- li[-1]
		- return in reverse order
	- li2 = li[:] # => li2 = [1,2,4,3] but (li2 is li) will result in false **(shallow copy)**

**Insert/Remove/Extend**
*Insert*

*Remove*
- del li[2] **remove index**
	- universal for set, hashmap, etc, to remove at indices

- li.remove(2) # **removes value**
- li.insert(1,2) # insert at index 1, value 2 

- li.index(2) # find index of val 2
- li.index(4) # if dne, raise ValueError

*Extend*
- li + other_li # creates new variable/object
- li.extend(other_li) # adds to same first object
