# Intro
- for multiple variables other than just X and Y
- this time we have multiple predictors X
- p many coefficients

- just like linear regression, we want the RSS
- how do we calc RSS? by finding Betas
- how do we find Betas? ->
	- instead of Covariance of X,Y, we have to do X1Y, X2Y, ...
	- thus, we format it into a matrix format so we can calculate it easier/faster

## Estimating Coefficients
- Y = XB + eps
```
Y = [y1]
	[y2]
	[...yn]
size(Y) = n*1

X = [1  x11  x12  ...  x1p]
	[1  x21  x22  ...  x2p]
	[... ... ...  ...  ...]
	[1  xn1  xn2  ...  xnp]

B = [B0]
	[B1]
	[...Bp]
size(B) = n*1 
```
- first, format X matrix
- last, get Beta B value

### (cont.) - compute X, Y, calculate B
yi = B0 + B1 x_i1 + ... + Bo x_ip + eps_i
- Compute correlations among predictors
	- X^T X
- Compute correlations b/w predictors and the response
	- X^T Y
- Express B in terms of these correlations

- so in linear regression, you have Cov(X,Y)
- in multiple linear regression, you have the correlation

### (cont.) - RSS
- ith residual: epsi = yi - ^yi = yi - x^T\_i B
```
eps = Y - XB = [y1-^y1]
			   [y2-^y2]
			   [...yn-^yn]
size -> (n*1)
so transpose of matrix is (1*n)

RSS = (Y-XB)^T (Y-XB) = SUM(i to n) [yi-^yi]^2
```

### (cont.) Least Squares method
- Least Squares Method requires:
	- min_B(SUM(i to n)[yi-^yi]^2) = min_B(Y-XB)^T (Y-XB)

- partial derivative:
	- partialL/partialB = 0 - 2X^T Y + 2X^T XB + 2 (X^T XB - X^T Y)
		- differentiating of matrix is its identity matrix (or 1)

- Solve for B:
	- partialL/partialB = 2(X^T XB) (X^T Y)
- setting partialL/partialB = 0, we have:
	- ^B = (X^T X)^-1 (X^T Y)

## Ex:
- cols: soap, soap^2, and suds

### compute X^T X and X^T Y
```
X^T X = 3x3 matrix
- 
X^T Y = 3x1 matrix
- how each predictor is related to Y_response
```
### find B via inverse matrix
```
^B = (X^T X)^-1 * (X^X Y)
-> 1/ () * matrix[ ]

^B = [3x1 matrix]

so our regression model is:
suds = -34.7143 + 21.5476 * soap - 1.09524 * soap^2
```

### find RSS
```
predictions using our model:
^y = 7x1 vector, eps = 7x1, RSS ~= 13.6667
```
