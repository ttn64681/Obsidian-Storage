# Dual Interface
- a potentially confusing feature of Matplotlib
	1. a convenient MATLAB-style state-based interface
		- simple, and default method
	2. a more powerful object-oriented interface
		- more convenient

## MATLAB-style Interface
- good for simple plotting
```
plt.figure() # create plot fig

plt.plot(x, np.sin(x)) # x and y axis

plt.plot(x, np.cos(x))
```
- figure is og
- plot() plots on same og figure

## Object-oriented Interface
### Multiple subplots
- good for simple plotting
```
fig, ax = plt.subplots(2) # fig (2d), and axes object (subplots)
# param - grid of (2) subplots arranged in 2 rows and 1 col

ax[0].plot(x, np.sin(x), 'r') # first axis, first subplot (red line)
ax[1].plot(x, np.sin(2*x)) 

plt.plot(x, np.cos(x))
```
- figure is og
- plot() plots on same og figure

**.subplots(#)**
- Used when you want to create a **regular grid of subplots** (e.g., 2 rows, 1 column, or a 2x2 grid).
- `plt.subplots()` automatically handles the positioning and spacing of these subplots.

# Histograms, Binnings, Densities
- rng = np.random.default_rng(1701)
- data = rng.normal(size=1000)
- plt.hist(data);

```
plt.hist(data, bins=30, density=True, alpha=0.5,
histtype='stepfilled', color='steelblue', edgecolor='none');
```

The type of histogram to draw:
- 'bar' is a traditional bar-type histogram. If multiple data are given the bars are arranged side by side.
- 'barstacked' is a bar-type histogram where multiple data are stacked on top of each other
- 'step'  generates a lineplot that is default unfilled
- 'stepfilled' is default filled

for function parameters, when passing in kwargs, you pass it in via **

```
x1 = rng.normal(0, 0.8, 1000)
x2
x3

kwargs = dict(histtype='stepfilled', alpha=0.3, density=True, bins=40)

plt.hist(x1, **kwargs)
plt.hist(x2, **kwargs)
plt.hist(x3, **kwargs)
```

## Multiple Subplots
- Used when you need to **manually position a subplot** at an arbitrary location on the figure.
```
fig = plt.figure()
ax1 = fig.add_axes([..,..,..,..,..], xticklabels=[], ylin=())
ax2 = fig.add_axes()
x= np.linspace(0,10)

ax1.plot(np.sin(x))
ax2.plot(np.cos(x))
```


## plt.subplot

```
for i in range(1,7):
	plt.subplot(2,3,i)
	plt.text(.5,.5, str((2,3,i))),
		fontsize=18, ha='center')
```
