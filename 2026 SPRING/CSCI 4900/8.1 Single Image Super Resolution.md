# Single image Super Resolution
- if you have 480 x 480 px image res, and you want 4k image res,
- then you need this super resolution technique
- this is used in image, video, and gaming resolution

## Image classification + Super-res problem
- if we train an image-classification model on CIFAR-10, and try it on different dataset, won't work as well
    
- for super-resolution, the task is generating a higher-resolution image, rather than classifying its content.
	- but if datasets vary differently, like real world images vs anime images, then it most likely won't work that well
    
- for image-classification, however, directly applying a model trained on a vastly different dataset typically performs poorly.
    
- most common strategy is training your models on training set of DIV2K dataset

## Image Quality Assessment (IQA)
- **PSNR** (pixel signal )
	- metric people use to measure performance
	- the higher the better

- what kind of perceptual quality we need (e.g., more realistic images, more clearer images, ...)

## Super-resolution application
- Nvidia - DLSS
- Deep Learning Super Sampling
	- DLSS 2: Super Resolution
		- **Problem:** can't render whole thing since will be slower, and frames will drop
		- **Solution:** only render a smaller portion of thing and upscale it
	  DLSS 3 (after RTX 40 series) : Super Resolution + Frame Generation
		- Predict frames directly
		- Predict 1st and 5th frame, and 2,3,4th frames are generated
	- DLSS 3.5: Super Resolution + Frame Generation + Ray Reconstruction

		- Directly using AI to signal the light conditions
