# GAN 
- generator model
	- used to generate something (image, sentence)
	- output is a matrix/vector (image or sentence)
- discriminator model
	- generates a scalar, representing how real/high quality image is
	- is it a real image or neural network image?

## Basic Idea of GAN
### Generator   
- vector -> Generator (it is a neural network, or a function) -> Matrix/vector
### Discriminator
- takes image as input -> Discriminator -> scalar

## Adversarial Training
- The Generator and Discriminator are trained simultaneously in an adversarial game.

- **Generator's Goal**: To produce data convincing enough to fool the Discriminator.
- **Discriminator's Goal**: To accurately identify whether a given data sample is real or fake.

- As the Discriminator becomes better at identifying fake data, it forces the Generator to improve its output quality to become more realistic.

- This dynamic can be likened to a "Prey and Predator" relationship, where the Generator (prey) continuously evolves to escape the Discriminator (predator).

Ex:
1. NN Generator v1 -> images -> Discriminator v1
2. NN Generator v2 -> images -> Discriminator v2

Another version of explanation:
1. NN Generator v1 -> images -> Discriminator v1 says "No eyes"
2. NN Generator v2 -> images -> Discriminator v1 says "No mouth"
3. NN Generator v3 -> images -> Discriminator v1 says "...."

## Training Algorithm:
- Initialize generator and discriminator G D
- in each training iteration:

- Learning D
	- sample 4 objects (code: 1 1 1 1)
	- generate 4 objects (code: 0 0 0 0)

	- discriminator updates
	- code code code code  generator fixes

	- this is similar to classification

	- discriminator produces score, and the generator's goal is to maximize the score from discriminator

- Learning G
	- code -> G -L images -> D -> 1
