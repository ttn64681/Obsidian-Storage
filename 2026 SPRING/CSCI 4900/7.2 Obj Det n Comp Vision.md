# Human Pose Estimation
- drawing skeletons

- Topdown method (e.g., AlphaPose)
	1. Utilize Object detectors to provide initial human crops first
	2. Network produces human keypoints in each cropped object
		- finding person, grab bounding box, then combine, thebn draw skeleton
		- similar to 2 stage object detection
		- first needs to have obj detection network
- Bottomup method (e.g., OpenPose)
	- directly drawing skeleton onto entire image
	- faster since draws first
	- difficult:
		- may detect multiple limbs, but needs to figure out which limbs belong to same person
	- multiperson system:
		- performs very well on multi-person
